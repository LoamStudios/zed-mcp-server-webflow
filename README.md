# Webflow MCP Server Extension for Zed

This extension integrates [Webflow MCP Server](https://github.com/webflow/mcp-server) as a context server for Zed's Assistant.


```
"context_servers": {
    "mcp-server-webflow": {
      "settings": {
        "webflow_token": "<WEBFLOW_TOKEN>"
      }
    }
  },
```
